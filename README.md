## このプロジェクトについて
このプロジェクトは、Web3層で構成する標準的なインターネット向けのWebアプリケーションを構築するために使用するテンプレートです。アプリケーションレイヤにはAWS Fargateを使用することにより、コンテナを用いてアプリケーションを構築することを想定しています。
